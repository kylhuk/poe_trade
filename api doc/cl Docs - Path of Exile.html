PoE2 EA - 0.4.0:

The Last of the Druids

  • Added passives→quest_stats to the Character type.

3.27.0:

Keepers of the Flame
  • Added the mutatedMods property to Item.
  • Added the mutated property to Item.
  • Added 13 (Breach skill frame) as possible values in the FrameType enum.
  • Added version 6.2 of the Passive Skill Tree URL which includes Bloodline information.
  • The PoE2 Character endpoint will no longer return inventory (unequipped) items.
Updated 28/10/2025.

PoE2 EA - 0.3.0:

The Third Edict
  • Added the Currency Exchange endpoint and associated scope.
  • The shapeshift key of the passives→specialisations property in Character has been renamed to set3.
  • Added the iconTierText property to Item.
  • Added the sanctified property to Item.
  • Added the desecratedMods property to Item.
  • Added the desecrated property to Item.
  • Removed the flavourTextParsed property from Item.
Updated 27/08/2025.

3.26.0:

Secrets of the Atlas
  • Updated the OAuth section to reference OAuth 2.1, which is essentially what our OAuth implementation already follows.
  • Clarified redirect URI requirements for each client type.
  • Added memoryItem to the Item type.
  • Removed category and subcategories from Item→extended.
Updated 11/06/2025.

PoE2 EA - 0.2.0:

Dawn of the Hunt
  • Added the poe2 realm option to the League endpoints. Ladders are still PoE1 only.
Updated 05/04/2025.

PoE2 Early Access
  • Added the poe2 realm option to the Character endpoints.
  • Updated the Character type to have new PoE2-specific properties.
  • Updated the Item type to have new PoE2-specific properties.
  • Updated the ItemSocket type to have new PoE2-specific properties.
  • Updated the ItemProperty type to have new PoE2-specific properties.
  • Added the PoE2-specific GemTab type.
  • Added the PoE2-specific GemPage type.
Updated 28/03/2025.

PoE2 EA - 0.1.1:

PoE2 Early Access
  • Clarified that there are currently no APIs that return PoE2 game information.
  • Updated the realm properties in the Create Item Filter and Update Item Filter endpoints to support poe2.
Updated 16/01/2025.

3.25.2:

Shared Account Patch
  • All account names now have a 4 digit discriminator suffix. Example: Novynn#0000.
  • Added an optional realm parameter to the Account Leagues endpoint.
  • Added an optional realm parameter to the Account Characters endpoint.
  • Added an optional realm parameter to the Account Stashes endpoint.
  • Added an optional realm parameter to the League Accounts endpoint.
  • Added an optional realm parameter to the Guild Stashes endpoint.
Updated 14/11/2024.

3.25.0:

Settlers
  • Added the Guild Stashes endpoint and associated scope. We will only grant this to a limited amount of applications to ensure service stability.
  • Marked category and subcategories from Item→extended as deprecated. It will be removed in a future version.
Updated 25/07/2024.

3.24.0:

Necropolis
  • Added atlas_passive_trees to the LeagueAccount type.
  • Added 11 as a possible value in the FrameType enum.
Updated 27/03/2024.

3.23.0:

Affliction
  • Added category to the League type.
  • Added rarity to the Item type.
  • Split out ItemProperty→displayMode into it's own type definition: DisplayMode.
  • Added 4 as a possible value in the DisplayMode enum.
  • Added rucksack to the Character type.
  • Added passives→alternate_ascendancy to the Character type.
  • Added version 6.1 of the Passive Skill Tree URL which includes Wildwood Ascendancy information.
Updated 08/12/2023.

3.22.0:

Trial of the Ancestors
  • Added a new Data Exports section clarifying the data available outside of APIs.
  • Added the Account Leagues endpoint and associated scope.
  • Split out Item→frameType into it's own type definition: FrameType.
  • Added rewards to the Item type. This represents rewards on itemised Sanctums.
  • Added foreseeing to the Item type. This will appear on items that have Foresight from Hinekora's Lock.
  • Added realm and ruthless to the Character type.
  • Added metadata and metadata→version to the Character type. This is the current game version of the character's realm.
  • Added passives→skill_overrides to the Character type. This will be used to provide Tattoo data.
  • Added isTattoo to the PassiveNode type.
  • Updated most sections to include self-links for easier linking.
Updated 16/08/2023.
